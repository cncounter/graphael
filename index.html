gRaphael—JavaScript 图表绘制工具

 

 简介

gRaphael’s goal is to help you create stunning charts on your website. It is based on Raphael graphics library. Check out the demos to see static and interactive charts in action.

gRaphael currently supports Firefox 3.0+, Safari 3.0+, Opera 9.5+ and Internet Explorer 6.0+.

 How to use it?

下载后在 HTML页面中按顺序引入 raphael.js , g.raphael.js 以及需要的 g.line.js , g.bar.js , g.dot.js 和 g.pie.js , 就可以像下面这样使用了: 

// 创建一块画布(canvas,在 Raphael 中一般称为 paper,画纸)
// 左上角坐标点: 10, 50, 宽高为 640 × 480 
var r = Raphael(10, 50, 640, 480);
// 以坐标 320, 200 为中心点, 创建饼图
// 饼图的半径为 100， 使用数据: [55, 20, 13, 32, 5, 1, 2]
r.piechart(320, 240, 100, [55, 20, 13, 32, 5, 1, 2]);
